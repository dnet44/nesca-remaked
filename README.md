Nesca (Modernized)
----
Сканер сети. Легендарный. Твой. Теперь быстрее и современнее.

![Nesca](examples/old_nesca.png)

### Что нового?

Программа была значительно переработана в соответствии с современными стандартами C++:

*   **Переход на C++17**: Использование современных возможностей языка для повышения безопасности и производительности.
*   **Пул потоков (Thread Pool)**: Вместо создания нового потока на каждый запрос, теперь используется эффективный пул воркеров. Это снижает накладные расходы и предотвращает перегрузку системы.
*   **Оптимизация алгоритмов**:
    *   Выборка IP-адресов теперь происходит за **O(1)** вместо O(N).
    *   Парсинг HTTP-заголовков оптимизирован с помощью `std::string_view`, что минимизирует лишние аллокации памяти.
*   **Потокобезопасность**: Глобальные счетчики переведены на `std::atomic`, что обеспечивает корректную работу в многопоточной среде.
*   **Удаление задержек**: Убраны искусственные ограничения скорости (Sleep), мешавшие полной утилизации канала.

### Самостоятельная сборка

Для компиляции необходимы `libssh`, `openssl`, `libcurl` и `Qt5`. Проект использует стандарт **C++17**.

Установка необходимого под Ubuntu:

1. Пакеты зависимостей:

```bash
sudo apt-get update
sudo apt-get install qt5-qmake qtbase5-dev qtmultimedia5-dev libqt5multimediawidgets5 \
                     libqt5multimedia5-plugins libqt5multimedia5 libssh-dev libcurl4-openssl-dev
```

Сборка:

```bash
qmake nesca.pro
make -j$(nproc)
```

### Сборка с помощью Docker

Можно сбилдить с помощью контейнера на основе Ubuntu, достаточно одной команды в директории с репозиторием:
```bash
docker build .
```

Для запуска Nesca прямо из контейнера выполнить:
```bash
docker run -i -e DISPLAY=$DISPLAY -v /tmp/.X11-unix/:/tmp/.X11-unix <IMAGE ID>
```
